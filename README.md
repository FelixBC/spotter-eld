# Spotter ELD

Trip planner for property-carrying truck drivers that auto-generates FMCSA-compliant ELD daily logs and route maps.

**Status:** 🚧 In development

## Stack

- **Backend:** Django + DRF
- **Frontend:** React + Vite + Tailwind
- **Maps & routing:** Leaflet + OpenRouteService
